# LoveSync

O LoveSync é uma plataforma desenvolvida para ajudar casais a organizarem sua rotina, finanças, metas e memórias em um único lugar.

Projeto full stack focado em autenticação JWT, WebSocket, upload de arquivos, dashboard e integração entre usuários.

## 🎯 Funcionalidades Principais

- **Autenticação JWT**: Sistema seguro de login e autenticação
- **WebSocket**: Comunicação em tempo real entre usuários
- **Upload de Arquivos**: Compartilhamento de fotos e documentos
- **Dashboard**: Visualização centralizada de dados do casal
- **Integração entre Usuários**: Sincronização de informações em tempo real

## 🛠️ Tecnologias

- Full Stack Development
- JWT Authentication
- WebSocket
- File Upload
